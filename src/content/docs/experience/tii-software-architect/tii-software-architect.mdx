*Abu Dhabi, UAE · Hybrid · Contract · Jun 2025 – Mar 2026*

- Owned the architecture of an internal platform rebuild while remaining hands-on within an agile team. This included the team's first ASP.NET Core Web API platform on .NET 9, structured around clean architecture, 50+ versioned endpoints, 30+ domain entities, and seven SAP integrations maintained through resilient HTTP clients and scheduled background jobs.
- Designed the event-driven workflow engine that supports multi-step approvals. The implementation used a finite state machine in which every transition is published through Wolverine's durable, SQL-backed queues with inbox/outbox semantics, ensuring that approvals, notifications, and emails can survive restarts and retry cleanly.
- Built the platform's event management and notification stack so operational communication could be handled consistently across the system. This included Fluid-rendered email templates dispatched from workflow events, and a normalised application-error event that carries correlation IDs through the message pipeline before handing off to email, Teams, or a future monitoring sink without changing event producers.
- Supported the migration away from a legacy WebForms application through a strangler-fig approach. In order to minimise disruption, the bridge was packaged as a backward-compatible .NET Standard 2.0 NuGet library so existing consumers could continue to operate while functionality moved into the new platform.
- Introduced delivery controls where no formal version control or CI/CD pipeline existed. GitLab was set up with build, test, and publish stages, JUnit-reported test runs, and artifact promotion across environments; structured logging and health checks were then added so production behaviour could be traced with greater confidence.
- Reviewed every merge request, led technical design sessions, and introduced agentic coding tools into the team's daily workflow. Claude Code, OpenAI Codex, and GitHub Copilot were applied to scaffolding, test generation, and refactoring in order to improve engineering throughput while preserving architectural oversight.
