*London, UK · Remote · Full-time · Aug 2019 – May 2024*

- Designed and delivered full-stack SaaS capabilities across C#/.NET services, TypeScript frontends, Angular 17, REST APIs, WebSocket APIs, data pipelines, and customer-facing analytics dashboards.
- Ran a performance programme on a 1.6 billion tweet Elasticsearch cluster after months of degrading search. More than 20 targeted fixes reduced JVM memory pressure by 15%, search threadpool rejections by 99%, average search latency by 89%, application costs by 92%, and API requests by 50%.
- Increased Search API uptime to 100% and reduced CPU usage from roughly 90% to roughly 10% by profiling production behaviour, reproducing failures, and applying focused C#/.NET optimisation work.
- The platform suffered site-wide outages several times a day. Redesigned the weakest ingestion and processing paths around producer/consumer queues, multithreading, TPL Dataflow, and concurrency controls, virtually eliminating downtime and earning a company share award.
- Found and fixed a five-year-old defect in the core Twitter ingestion pipeline that had silently capped data yield at 6.25% per request. Restoring it to 100% lifted search, reporting, analytics, and profile quality across every product.
- Led the Twitter API v1-to-v2 migration across 10+ dependent applications by analysing dependencies, creating Azure DevOps epics and issues, redesigning rate-limit handling, and choosing an adapter strategy so teams could move without downstream disruption.
- Delivered YouTube and social-data ingestion pipelines that refreshed 70,000+ accounts daily and supported millions of posts, using research, prototyping, priority-queue scheduling, API-limit-aware backfills, and production monitoring.
- Expanded notification, ingestion, and reporting systems using AWS CloudFormation, Lambda, SNS, SQS, SES, S3, CloudWatch, scheduled triggers, REST endpoints, email pipelines, Kinesis streams, DynamoDB Streams, and Elasticsearch upload paths.
- Refactored legacy analytics delivery logic with TPL Dataflow, OOP, and SOLID patterns, reducing data consolidation time by 99% and achieving a 360x efficiency increase.
- Improved data retrieval performance from DynamoDB by 1,000% over the AWS SDK and increased retrieval performance by 16,000% on an existing product feature.
- Built and deployed an end-to-end tweet classifier handling 125M records a month, owning exploratory data analysis, feature engineering, classifier selection, model evaluation, production delivery, and monitoring.
- Strengthened quality and release practices with xUnit, Moq, Fluent Assertions, AutoFixture, JavaScript linting, Husky pre-commit checks, CloudFormation updates, CodePipeline, CodeBuild, GitHub code reviews, developer documentation, architecture notes, debugging guides, and log-analysis practices.
- Reviewed 500+ pull requests across 30 repositories, mentored engineers on SOLID, DDD, API design, microservices, AWS, Docker, testing, and built 150+ custom GPT assistants for engineering, architecture, refactoring, and TDD support.
- Collaborated with project managers, business analysts, architects, designers, stakeholders, and client teams to split work, estimate complexity, document dependencies, support design decisions, and build from common ground toward product outcomes.
