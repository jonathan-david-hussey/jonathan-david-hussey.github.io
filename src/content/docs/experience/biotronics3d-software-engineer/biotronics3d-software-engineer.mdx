*Edinburgh, UK · On-site · Full-time · Nov 2017 – Aug 2019*

- Designed a generic real-time communications framework for a radiology platform — one foundation that ended up powering chat, live notifications, data synchronisation and VOIP between radiologists.
- Concurrent edits to patient records were a real safety concern, so I built conflict resolution and consolidation logic that let multiple staff modify appointments simultaneously without losing data.
- Architected and integrated WebRTC-based voice calling so remote radiologists could collaborate on patient scans directly inside the product.
- Contributed across a new Radiology Information System and its integration with the existing PACS imaging system, owning features end to end from design through deployment.
